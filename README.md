# Easy-Script-Manager
A simple script manager for live video
## Easy Script Manager(ESM)とは
私(qwe0412_)がAIツールを用いて作成した実況動画等の台本を簡単に管理できるソフトウェアです。
## ESMの使い方
基本的にはReleasesに投稿していますが、もしかすると他の配布形態に変更するかもしれません。
## 仕組み
基本的にはpythonで作られている~~ゴミ~~ソフトウェアです。
.esmでプロジェクトを保存でき、テキストへのエクスポートも対応しています。そのうち他のファイルに対応するかも。
